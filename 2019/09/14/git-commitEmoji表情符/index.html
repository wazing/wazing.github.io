git commitEmoji表情符

发表于 2019-09-14 | 分类于 Android | 阅读次数:

在向github提交commit的时候插入emoji表情符，但是emoji表情符在提交代码的时候也不能乱用，否则容易造成误解。因此开源项目 gitmoji) 规定了在 github 提交代码时应当遵循的 emoji 规范:

Emoji 代码 规则 Description
🎨 :art: 改进代码的结构/格式 Improving structure / format of the code.
⚡️ :zap: 提高性能 Improving performance.
🔥 :fire: 删除代码或文件 Removing code or files.
🐛 :bug: 修复bug Fixing a bug.
🚑 :ambulance: 修补关键程序 Critical hotfix.
✨ :sparkles: 引入新功能 Introducing new features.
📝 :pencil: 编写文档 Writing docs.
🚀 :rocket: 部署项目 Deploying stuff.
💄 :lipstick: 更新UI和style文件 Updating the UI and style files.
🎉 :tada: 第一次提交 Initial commit.
✅ :white_check_mark: 更新测试用例 Updating tests.
🔒 :lock: 解决issues问题 Fixing security issues.
🍎 :apple: 在macOS上修复问题 Fixing something on macOS.
🐧 :penguin: 在Linux上修复问题 Fixing something on Linux.
🏁 :checkered_flag: 在Windows上修复问题 Fixing something on Windows.
🤖️ :robot: 在Android上修复问题 Fixing something on Android.
🍏 :green_apple: 在iOS上修复问题 Fixing something on iOS.
🔖 :bookmark: 发布版本/拉取tag Releasing / Version tags.
🚨 :rotating_light: 修复lint警告 Removing linter warnings.
🚧 :construction: Work in progress.
💚 :green_heart: Fixing CI Build.
⬇️ :arrow_down: 降级依赖性 Downgrading dependencies.
⬆️ :arrow_up: 升级依赖关系 Upgrading dependencies.
📌 :pushpin: 将依赖项固定到特定版本 Pinning dependencies to specific versions.
👷 :construction_worker: Adding CI build system.
📈 :chart_with_upwards_trend: 添加分析或跟踪代码 Adding analytics or tracking code.
♻️ :recycle: 重构代码 Refactoring code.
🐳 :whale: Work about Docker.
➕ :heavy_plus_sign: 添加依赖项 Adding a dependency.
➖ :heavy_minus_sign: 删除依赖项 Removing a dependency.
🔧 :wrench: 更改配置文件 Changing configuration files.
🌐 :globe_with_meridians: 国际化和本地化 Internationalization and localization.
✏️ :pencil2: 修正错别字 Fixing typos.
💩 :poop: Writing bad code that needs to be improved.
⏪ :rewind: 代码回退 Reverting changes.
🔀 :twisted_rightwards_arrows: 合并分支结构 Merging branches.
📦 :package: 更新编译的文件或包 Updating compiled files or packages.
👽 :alien: Updating code due to external API changes.
🚚 :truck: 移动或重命名文件 Moving or renaming files.
📄 :page_facing_up: 添加或更新license Adding or updating license.
💥 :boom: 引入重大变化 Introducing breaking changes.
🍱 :bento: 添加或更新Assets Adding or updating assets.
👌 :ok_hand: Updating code due to code review changes.
♿️ :wheelchair: 改善可访问性 Improving accessibility.
💡 :bulb: 记录源代码 Documenting source code.
🍻 :beers: Writing code drunkenly.
💬 :speech_balloon: 更新文本和文字 Updating text and literals.
🗃 :card_file_box: 执行数据库相关的更改 Performing database related changes.
🔈 :loud_sound: 添加日志 Adding logs.
🔇 :mute: 删除日志 Removing logs.
👥 :busts_in_silhouette: 添加贡献者 Adding contributor(s).
🚸 :children_crossing: 改善用户体验/可用性 Improving user experience / usability.
🏗 :building_construction: 进行架构更改 Making architectural changes.
📱 :iphone: Working on responsive design.
🤡 :clown_face: Mocking things.
🥚 :egg: 添加一个彩蛋 Adding an easter egg.
🙈 :see_no_evil: 添加或更新.gitignore文件 Adding or updating a .gitignore file
📸️ :camera_flash: 添加或更新快照 Adding or updating snapshots
⚗️ :alembic: 新事物实验 Experimenting new things
🔍 :mag: 改进SEO Improving SEO
☸️ :wheel_of_dharma: 在Kubernetes上工作 Work about Kubernetes
🏷 :label: 添加或更新类型（Flow，TypeScript） Adding or updating types (Flow, TypeScript)
🌱 :seedling: Adding or updating seed files

END
