使用Jsoup解析HTML

发表于 2019-05-31 | 分类于 Android | 阅读次数:

Jsoup 是一款 Java 的 HTML 解析器，可直接解析某个 URL 地址、HTML 文本内容。它提供了一套非常省力的 API，可通过 DOM，CSS 以及类似于 jQuery 的操作方法来取出和操作数据。

jsoup 的主要功能如下：

  1. 从一个 URL，文件或字符串中解析 HTML；

  2. 使用 DOM 或 CSS 选择器来查找、取出数据；

  3. 可操作 HTML 元素、属性、文本；
